# Employee-management-system

<h2>PROJECT DESCRIPTION</h2>
A responsive Employee Management System built using HTML, CSS, and JavaScript.  
This application allows users to add, edit, delete, and view employees with pagination support.  
Data is stored using browser Local Storage.

<h2>Features</h2>

- Add Employee
- Edit Employee
- Delete Employee (with confirmation)
- View Employee List
- Pagination (5 records per page)
- Data persistence using Local Storage
- Responsive Design
